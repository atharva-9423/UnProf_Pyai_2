## ✨ What Can It Do?

| Feature | Description |
|---|---|
| ➕ Add Contact | Save a contact's name, phone number, and email |
| 🔍 Search Contact | Find anyone by their name, phone, or email |
| 📋 View All Contacts | See everyone saved in your contact list |
| 💾 Auto Save | Contacts are saved to a file — no data is lost when you close the app |

---

## 🚀 How To Run

**Step 1** — Make sure Python is installed on your computer
> Download Python from [python.org](https://python.org) if you don't have it

**Step 2** — Open a terminal in the project folder and run:

```bash
python3 contact_manager.py
```

**Step 3** — Use the menu that appears:

```
===== MENU =====
1. Add Contact
2. Search Contact
3. View All Contacts
4. Exit
```

Just type a number and press Enter — that's it!

---

## 💡 How It Works (Simply Explained)

- When you **add a contact**, it gets saved into a file called `contacts.json` on your computer
- When you **open the app again**, it reads that file and loads all your contacts back
- When you **search**, it checks the name, phone, and email all at once — so you only need to remember one detail
